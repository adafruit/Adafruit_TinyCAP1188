This is a library for the CAP1188 8-channel Capacitive touch sensor on the Trinket or Gemma

Designed specifically to work with the CAP1188 Breakout in the Adafruit shop 
  ----> https://www.adafruit.com/products/

On the Tinket/Gemma these sensors use I2C to communicate, pins 0 and 2 are required to interface

Adafruit invests time and resources providing this open source code,

Check out the links above for our tutorials and wiring diagrams 

To download. click the ZIP button, and rename the uncompressed folder TinyCAP1188 
Check that the TinyCAP1188 folder contains TinyCAP1188.cpp and TinyCAP1188.h

Place the TinyCAP1188 library folder your arduinosketchfolder/libraries/ folder. 
You may need to create the libraries subfolder if its your first library. Restart the IDE.

We also have a great tutorial on Arduino library installation at:
http://learn.adafruit.com/adafruit-all-about-arduino-libraries-install-use
